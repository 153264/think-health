# CheckFolder 文件夹大小检查器

## 实例化

```php
new \think\health\Check\CheckFolder(
    /**
     * 需要检查的文件夹
     * 键为文件夹路径，值为允许的最大大小（字节）
     * @var array<string,int>
     */
    folders: [
        runtime_path() => 1024 * 1024 * 1024
    ]
)
```

## 参数说明

| 参数    | 类型                | 是否必传 | 默认值 | 说明                                                              |
| ------- | ------------------- | -------- | ------ | ----------------------------------------------------------------- |
| folders | array\<string,int\> | 是       | -      | 需要检查的文件夹<br/>键为文件夹路径<br>值为允许的最大大小（字节） |
